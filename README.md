# retail-sales-performance

## Overview
Interactive Excel dashboard analyzing sales performance by store, department and time.

## Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables & Charts
- Slicers

## Data
- Retail Dataset from Kaggle: https://www.kaggle.com/datasets/manjeetsingh/retaildataset/data 
- Cleaned and transformed using Power Query
